Projects Security Door

Project Description

This project is called AKSYS, which is an acronym for Automatic Keyless System. This system was created to solve the problem for an economic, secure, convenient and easy to make convenience for unlocking and locking sliding bolt latch doors. For the system, we used the Arduino UNO device along with the Bluetooth, LED, and servo attachments. The device was programmed using the Arduino language(largely based on C/C++) and the Arduino IDE.

Project Details

  • Developed the prototype on Arduino Uno
  • Bluetooth communication between the Android phone and Arduino
  • Developed GUI application for user-friendly to control servo

Projects Phone Book

Project Description

This assignment, I created the data structure of the yellow pages. It implemented the doubly linked list with node (head and tail) and data. Each data has a dictionary node with the key is a phone number and the value is the name of the owner. It stored the data in each hashtable, binary search tree, and red-black tree for insertion, deletion, and search

Project Details

  • Created the data structure doubly linked list and dictionary node
  • Stored doubly linked list in the hashtable, binary search tree, and red-black tree
  • Insertion, Deletion, and Search the dictionary node correspondence with the complexity big O of 1 and log(n) - O(1) and O(log(n))
