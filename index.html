Projects Security Door

Project Description

This project is called AKSYS, which is an acronym for Automatic Keyless System. This system was created to solve the problem for an economic, secure, convenient and easy to make convenience for unlocking and locking sliding bolt latch doors. For the system, we used the Arduino UNO device along with the Bluetooth, LED, and servo attachments. The device was programmed using the Arduino language(largely based on C/C++) and the Arduino IDE.

Project Details

  • Implemented Android GUI application in the embedded system (Arduino - Microcontroller) to control servo
  • Bluetooth communication between the Arduino Uno and Android phone
  • Design and learning the Arduino circuit with a breadboard

Projects File Editor

Project Description

This assignment, I created multiple processes to handle the file editor. It is similar as text editor with an extra feature of backup file to prevent the loss of data.

Project Details

  • Implemented ​multiple processes​ ​(fork)​ for a file editor
  • Provided user multiple options to create, read, and write a file or folder (directory) using ​a child process (fork)
  • Implemented a backup file to prevent the loss of data due to a natural disaster or human errors
  • Implemented a quicksort algorithm to sort the content of the file in the alphabet order and the reverse order

Projects Timer Control

Project Description

The project designs and implements a programmable event with C/C++ language and timer control program. The program has two child threads. A thread monitors the countdown timer and notifies signal() or pipe(). Other child thread receives the notification and terminates a thread monitor and itself

Project Details

  • Created one ​process (fork)​ and two ​threads​ (pthread library) in the Linux kernel operating system
  • Implemented ​Makefile​ with shell script command for ​compiling​, ​linking library​, and ​build​ the C file
  • Implemented one thread "wall clock" which prints the hour, minute in local time
  • Implemented other thread received notification and terminate the program
